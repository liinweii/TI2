# TI2
## Лабораторная работа на тему потоковое шифрование. Многочлен 27 степени
Вариант 5

## О проекте
Программа для шифрования и дешифрования файлов различных расширений и размеров с помощью ключей, сгенерированных на основе сдвиговых регистров с линейной обратной связью LSFR(Linear Feedback Shift Register).

### Ссылка на программу: https://github.com/liinweii/TI_1/blob/main/lab1it.exe

# 📘 Инструкция по использованию программы
## Шаг 1. Загрузите файл для шифрования
Нажмите на кнопку "Load File" и выберите файл для шифрования.
Программа самостоятельно подсчитает количество бит в будущем ключе.

## Шаг 2. Заполните регистр сдвигов и сгенерируйте ключ  
Заполните свободное поле значениями 1 и 0. Все остальные символы будут проигнорированы. Введено должно быть ровно 27 значений. 
После этого нажмите на кнопку "Generate Key". Программа сгенерирует ключ и выведет на экран его первые значения. 

## Шаг 3. Шифрование и дешифрование
Используйте кнопки под полями ключа и текста:

Кнопка "encrypt":	преобразует исходную последовательность бит в зашифрованную  
Кнопка "decrypt":	восстанавливает исходную последовательность бит из зашифрованной  
✅ После нажатия в свободном поле появится обработанная последовательность.

## Шаг 4. Сохранение результата
Нажмите "save": результат сохранится в выбранный вами файл.
